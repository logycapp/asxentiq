# Documentacion operativa

Esta carpeta agrupa la documentacion de operacion del proyecto.

## Como usarla

1. Leer `AGENTS.md` para conocer las reglas de trabajo.
2. Leer `MAPA_PROYECTO.md` para entender el contexto general.
3. Elegir la parte que corresponda:
   - `agentes/backend.md` para tareas de Laravel.
   - `agentes/frontend.md` para tareas de Angular.
   - `agentes/modulos/{modulo}/README.md` para tareas de un modulo especifico.

## Criterio de uso

- Usa esta documentacion para ubicar el alcance de cada cambio.
- Revisa solo la seccion relevante a la tarea.
- Si el modulo o flujo no esta documentado, marca `No determinado` antes de asumir.
