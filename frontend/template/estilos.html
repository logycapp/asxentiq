Notas de uso

  • Usar fondo oscuro para superficies principales y azul profundo para headers y sidebars.
  • Reservar el azul brillante para acciones activas, enlaces y estados de énfasis.
  • Mantener tipografía Inter para UI y JetBrains Mono para tiempos, códigos y métricas.
  • Evitar mezclar componentes de login y dashboard sin clase de página específica.
